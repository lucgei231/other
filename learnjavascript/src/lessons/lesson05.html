Lesson 05: Understanding Functions in JavaScript

In this lesson, we will learn about functions, how to define them, and how to invoke them.

What is a Function?

A function is a block of code designed to perform a particular task. It is executed when "something" invokes it (calls it).

Defining a Function

Functions can be defined using the following syntax:

function functionName(parameters) {
        // code to be executed
    }

Invoking a Function

To execute a function, you need to call it by its name followed by parentheses:

functionName();

Example

Here is a simple example of a function that adds two numbers:

function add(a, b) {
        return a + b;
    }

    console.log(add(5, 3)); // Outputs: 8

Exercise

Try creating your own function that multiplies two numbers and returns the result.
