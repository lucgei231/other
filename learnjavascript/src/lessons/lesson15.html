Lesson 15: Understanding Promises in JavaScript

In this lesson, we will learn about Promises, a powerful feature in JavaScript for handling asynchronous operations.

What is a Promise?

A Promise is an object that represents the eventual completion (or failure) of an asynchronous operation and its resulting value.

Creating a Promise

To create a Promise, you can use the following syntax:

const myPromise = new Promise((resolve, reject) => {
    // Asynchronous operation
});

Using Promises

Once a Promise is created, you can use the .then() method to handle the resolved value and .catch() to handle any errors:

myPromise
    .then(result => {
        console.log(result);
    })
    .catch(error => {
        console.error(error);
    });

Example

Here is a simple example of a Promise that resolves after 2 seconds:

const delay = (ms) => {
    return new Promise(resolve => setTimeout(resolve, ms));
};

delay(2000).then(() => {
    console.log('Executed after 2 seconds');
});

Exercise

Try creating your own Promise that simulates a network request and resolves with a message after a delay.

Conclusion

Promises are a fundamental part of modern JavaScript, allowing for cleaner and more manageable asynchronous code.
