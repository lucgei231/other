Lesson 02: Variables and Data Types

In this lesson, you will learn about variables and the different data types in JavaScript.

What are Variables?

Variables are containers for storing data values. In JavaScript, you can create a variable using the let, const, or var keywords.

Data Types

JavaScript has several data types:

  • String: Represents textual data.
  • Number: Represents numeric values.
  • Boolean: Represents true or false values.
  • Object: Represents complex data structures.
  • Array: A special type of object for storing ordered collections.

Example

Here is an example of declaring variables:

let name = "John"; // String
let age = 30; // Number
let isStudent = true; // Boolean

Exercise
